# Security Policy

Book Factory may later store project data, prompts, book concepts, user notes, and production results. These assets can be commercially sensitive.

## Current Status

The project is in the foundation phase and contains no application code yet.

## Security Principles

- Do not store ChatGPT credentials.
- Do not request OpenAI API keys for normal use.
- Do not send user project data to paid APIs.
- Treat exported book material as private user data.
- Keep local storage and backup behavior explicit.
- Document every integration that transfers data outside the application.

## Reporting

Security concerns should be documented in an issue or private communication channel once the project hosting process is finalized.
